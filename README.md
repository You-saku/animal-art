# Animal-Art
This is CLI tool used Rust.

## Requirements
```
$ cargo -V
cargo 1.67.1 (2023-04-20)

$ rustup -V
rustup 1.25.2 (2023-04-20) 

$ rustc -V
rustc 1.67.1 (2023-04-20)
```

## Setup
```
git@github.com:You-saku/animal-art.git
cd animal-art
cargo build
cargo run src/animals/capybara.txt
```
